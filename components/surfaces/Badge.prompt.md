Pill-shaped label / tag — streak counters, routine labels, status chips.

```jsx
<Badge tone="streak" icon={<span>🔥</span>}>23일차</Badge>
<Badge tone="dark" size="sm">우주인님 루틴</Badge>
<Badge tone="brand">시작하기</Badge>
```

Tones: `streak` (white pill with shadow + flame), `brand`/`mint` (green family), `dark` (overlay label on photos), `neutral`. Bold by default.
