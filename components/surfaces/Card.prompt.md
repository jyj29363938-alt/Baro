Rounded surface container used for nearly every grouped block in Baro.

```jsx
<Card>
  <h3>오늘의 스트레칭</h3>
</Card>

<Card tone="mint-dashed">나의 목표 — 이번엔 진짜 한다!</Card>
```

Tones: `default` (white + soft gray shadow), `flat` (hairline border, no shadow), `mint` (brand wash), `mint-dashed` (dashed mint — goal / share panels). Default radius 20px, padding 20px.
